# smart-shopping-tracker
SmartShopping is a simple web application built using Node.js and Express.js that serves a static frontend. It demonstrates basic server setup, routing, and static file handling.


---
## Live Demo
https://madhaviy-340.github.io/smart-shopping-tracker/

##  Features

- Add shopping items with name and price
- Delete items from the list
- Automatically calculate total spending
- Track remaining budget in real time
- Alert when budget is exceeded
- Live updates using AJAX (jQuery)
- Simple and clean UI

---

##  Tech Stack

- Frontend: HTML, CSS, JavaScript, jQuery
- Backend: Node.js, Express.js
- API: REST API
- Data Storage: In-memory array (temporary storage)
